1. Что такое переменная в Python?
В Python переменная - это именованная область памяти, в которой можно хранить значение. 
Переменные в Python создаются присваиванием значения имени. 
Например:

x = 10
y = "строка"

2. Что такое счётчик ссылок и зачем он используется?
Счётчик ссылок:
В Python счётчик ссылок – это механизм управления памятью, 
используемый для отслеживания количества ссылок на объект. 
Каждый объект в Python имеет свой счётчик ссылок, и когда этот счётчик достигает нуля, 
объект удаляется из памяти. Это автоматический механизм сборки мусора, который освобождает память, 
занимаемую объектами, которые больше не используются.

3. Какие основные типы данных есть в Python?
Основные типы данных в Python:

Числовые типы данных:

int (целые числа)
float (вещественные числа)
complex (комплексные числа)
Текстовые типы данных:

str (строки)
Логический тип данных:

bool (логические значения True или False)
Коллекции:

list (списки)
tuple (кортежи)
set (множества)
dict (словари)
Другие типы:

NoneType (тип None, представляющий отсутствие значения)

4. Как можно преобразовать данные из одного типа в другой?
Переменная в Python:

Явное преобразование (типизация):

Используется функции, такие как int(), float(), str(), 
чтобы явно преобразовать данные из одного типа в другой.

x = 10
y = str(x)  # Преобразование целого числа в строку
Неявное преобразование (автоматическое приведение типов):

Python может автоматически преобразовывать типы в некоторых случаях, например, 
при выполнении математических операций между разными типами данных.

a = 5
b = 2.0
c = a + b  # Результат будет вещественным числом
